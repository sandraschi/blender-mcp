# Blender MCP User Guide

## Getting Started

1. **Configure Blender Path**: Set the path to your Blender executable in the user configuration
2. **Test Connection**: Try a simple operation like creating a cube
3. **Explore Tools**: Use the available tools to build your 3D scenes

## Basic Workflow

### Scene Creation
```
1. Create basic geometry (blender_mesh)
2. Add materials (blender_materials)
3. Set up lighting (blender_lighting)
4. Position camera (blender_camera)
5. Render preview (blender_render)
```

### Character Animation
```
1. Create armature (blender_rigging)
2. Add mesh and skin (blender_mesh, blender_modifiers)
3. Set keyframes (blender_animation)
4. Test animation (blender_render)
```

### Game Asset Pipeline
```
1. Model asset (blender_mesh, blender_modifiers)
2. UV unwrap (blender_uv)
3. Create materials (blender_textures)
4. Export for engine (blender_export)
```

## Tool Categories

- **Mesh Operations**: Create and modify 3D geometry
- **Materials**: Apply colors, textures, and shaders
- **Lighting**: Set up professional lighting rigs
- **Animation**: Create movement and character rigs
- **Rendering**: Produce final images and videos
- **Import/Export**: Work with external file formats

## Common Tasks

### Quick Scene Setup
- Create a cube at origin
- Add a camera and light
- Set up basic materials
- Render a test image

### Furniture Creation
- Use blender_furniture tools for parametric furniture
- Customize dimensions and styles
- Apply realistic materials

### Environment Building
- Import reference models
- Create terrain and architecture
- Add atmospheric effects
- Set up HDRI lighting

## Troubleshooting

### Blender Not Found
- Verify executable path in configuration
- Check file permissions
- Try auto-detection feature

### Operations Timeout
- Increase timeout in user config
- Simplify complex scenes
- Check system resources

### Export Issues
- Ensure output directories exist
- Check file format compatibility
- Verify object selection

## Advanced Features

- **Physics Simulation**: Add realistic movement and collisions
- **Particle Effects**: Create fire, smoke, and dynamic elements
- **Procedural Textures**: Generate complex surface patterns
- **Character Rigging**: Build controllable character models

## Best Practices

1. **Start Simple**: Begin with basic shapes and build complexity
2. **Test Frequently**: Render previews to catch issues early
3. **Organize Scenes**: Use clear naming and hierarchies
4. **Optimize Performance**: Use appropriate render settings for your needs
5. **Version Control**: Save incremental versions of your work
